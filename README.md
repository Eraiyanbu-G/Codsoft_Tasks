# 🚀 Python Mini Projects Collection

This repository contains beginner-friendly Python projects built to practice programming fundamentals.

---

## 📂 Projects Included

### 1️⃣ 📝 To-Do App
A simple terminal-based task manager.

**Features:**
- Add tasks
- View tasks
- Mark tasks as done
- Delete tasks

▶ Run:
python todo.py

---

### 2️⃣ 🧮 Calculator
A basic calculator application built using Python.

**Features:**
- Addition
- Subtraction
- Multiplication
- Division

▶ Run:
python calculator.py

---

### 3️⃣ 🔐 Password Generator
A secure random password generator.

**Features:**
- Generates strong random passwords
- Customizable password length
- Uses letters, numbers, and special characters

▶ Run:
python password.py
